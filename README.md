# Portfólio Pessoal - Guilherme Henrique

## Sobre o Projeto

Este projeto foi desenvolvido como atividade prática da disciplina de Desenvolvimento Web, com o objetivo de aplicar conceitos de HTML, CSS e JavaScript na criação de um portfólio pessoal responsivo e funcional.

O site apresenta informações profissionais, formação acadêmica, projetos desenvolvidos e formas de contato, utilizando uma estrutura moderna e organizada.

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages

---

## Funcionalidades

- Navegação por menu
- Layout responsivo
- Tema claro/escuro
- Formulário de contato com validação
- Seção sobre mim
- Formação acadêmica
- Portfólio de projetos
- Links para GitHub e LinkedIn
